## 引用 
*   引用其实只是一个别称，而非对象
*   只能赋值一次，因此引用必须被初始化
 

## 指针 


* 赋值为空指针的时候建议使用 nullptr 而非 NULL
* 一定要初始化指针，否则会引起不可预料的错误
* 指针也可以用于条件表达式中：非零 == true
* void* 指针 该视角内存空间就是内存空间，不能访问内存空间所存的对象
