先埋坑

# 1.特征的种类

从数据格式来讲，一般有以下三种：

- 类别特征
- 数值特征
- 序列特征

从场景划分角度来讲，一般可以分为：

- context feature
- user feature 
- item feature

从时效性角度来讲，也可以划分为：

- 长期特征
- 短期特征
- 实时特征



# 2.特征预处理

- 如何对特征进行处理？
- 是否需要进行归一化？
- 特征如何进行embedding？
